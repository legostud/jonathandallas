Pure CSS

As a learning exercise, I wanted to see how far I could push CSS to create graphics. As a starting point, I found a Flat Design of a classroom and then tried to copy it using just CSS. By leveraging linear and radial gradients and a heaping of transforms, I was able to create, what I think was, a pretty good match while adding some extra elements like a clock that is accurate to the local time and an interactive mouse.

Space Finder

One of the items requested for Harvard Library's website redesign was an app that students could use to find and book study rooms. The client had found an app at Cambridge University and asked if we could build something similar. Using React and CSS animmations, we were able to build a modern version while keeping a similar User Interface.

Data Visualizations

For PHCPI's latest Data Visualization, I decide to experiment to see if I could create these graphs using CSS instead of relying on D3. The plan was to make each graph a React App that received the color and value(s) to render. React would then convert those values into inline styles (max-width, positioning, background) where needed. All the graphs were doable with CSS, but we did use D3 to render the pie chart.
