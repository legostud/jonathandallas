Key New Jersey Programs and Initiatives

New Jersey Health Initiatives

New Jersey Health Initiatives (NJHI) supports advancement in health services delivery, improvement in health, and the expansion of leadership expertise through grantmaking across the state of New Jersey.

Learn more about this program

New Jersey Nursing Initiative

New Jersey Nursing Initiative (NJNI) addresses the state nurse faculty shortage by: developing a statewide model for recruiting and retaining nurse faculty; developing models for nursing curricula to educate new faculty; increasing the number of nurse faculty; and building partnerships among diverse stakeholders to support efforts to raise the number of nurse faculty.

Learn more about this program

New Jersey Partnership for Healthy Kids

New Jersey Partnership for Health Kids (NJPHK) is an RWJF statewide program, with technical assistance and direction by the New Jersey YMCA State Alliance. The program aims to convene, connect and empower community partnerships to implement environment and policy-changing strategies that prevent childhood obesity.
